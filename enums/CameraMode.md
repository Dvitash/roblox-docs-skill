# Enum.CameraMode

Used to set `Class.Player.CameraMode` to determine when first person and third person cameras should be used.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `Classic` | 0 |  | Third person camera which can be zoomed into first person. |
| `LockFirstPerson` | 1 |  | Exclusively first person camera. |

**Valid values:** `Enum.CameraMode.Classic`, `Enum.CameraMode.LockFirstPerson`
