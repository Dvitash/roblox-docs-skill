# Enum.PathStatus

The success of a `Class.Path` generated by `Class.PathfindingService`.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `Success` | 0 |  | Path found successfully. |
| `ClosestNoPath` | 1 | Deprecated | Path doesn't exist, returns path to closest point. |
| `ClosestOutOfRange` | 2 | Deprecated | Goal is out of MaxDistance range, returns path to closest point you can reach within MaxDistance. |
| `FailStartNotEmpty` | 3 | Deprecated | Failed to compute path; the starting point is not empty. |
| `FailFinishNotEmpty` | 4 | Deprecated | Failed to compute path; the finish point is not empty. |
| `NoPath` | 5 |  | Path doesn't exist. |

**Valid values:** `Enum.PathStatus.Success`, `Enum.PathStatus.ClosestNoPath`, `Enum.PathStatus.ClosestOutOfRange`, `Enum.PathStatus.FailStartNotEmpty`, `Enum.PathStatus.FailFinishNotEmpty`, `Enum.PathStatus.NoPath`
