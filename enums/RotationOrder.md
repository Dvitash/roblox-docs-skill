| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `XYZ` | 0 |  | X, Y, Z order. |
| `XZY` | 1 |  | X, Z, Y order. |
| `YZX` | 2 |  | Y, Z, X order. |
| `YXZ` | 3 |  | Y, X, Z order. |
| `ZXY` | 4 |  | Z, X, Y order. |
| `ZYX` | 5 |  | Z, Y, X order. |

**Valid values:** `Enum.RotationOrder.XYZ`, `Enum.RotationOrder.XZY`, `Enum.RotationOrder.YZX`, `Enum.RotationOrder.YXZ`, `Enum.RotationOrder.ZXY`, `Enum.RotationOrder.ZYX`
