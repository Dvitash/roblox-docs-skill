# Enum.CameraPanMode

The CameraPanMode Enum represents the available pan modes for `Class.Camera:SetCameraPanMode()`.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `Classic` | 0 |  | Enables swipe and pan on mobile devices, and disables edge bump camera controls. Default. |
| `EdgeBump` | 1 |  | Disables swipe to pan on mobile devices, and enables edge bump camera controls. It does not affect Windows or Mac users. |

**Valid values:** `Enum.CameraPanMode.Classic`, `Enum.CameraPanMode.EdgeBump`
