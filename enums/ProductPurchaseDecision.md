# Enum.ProductPurchaseDecision

The `ProductPurchaseDecisionEnum` works with `Class.MarketplaceService` to indicate the status of the sale of developer products.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `NotProcessedYet` | 0 |  | The purchase request has been sent but not yet processed. |
| `PurchaseGranted` | 1 |  | The purchase has been processed and granted to the user who initiated the purchase request. |

**Valid values:** `Enum.ProductPurchaseDecision.NotProcessedYet`, `Enum.ProductPurchaseDecision.PurchaseGranted`
