# Enum.TweenStatus

The completion status of a `Class.GuiObject` tween function.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `Canceled` | 0 |  | The tween was cancelled before completion. |
| `Completed` | 1 |  | The Tween has successfully completed. |

**Valid values:** `Enum.TweenStatus.Canceled`, `Enum.TweenStatus.Completed`
