# Enum.HorizontalAlignment

Used by `Class.UIGridStyleLayout.HorizontalAlignment` to align the layout horizontally within its parent.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `Center` | 0 |  | Grid is aligned to the center of its parent along the **X** axis. |
| `Left` | 1 |  | Grid is aligned to the left edge of its parent. |
| `Right` | 2 |  | Grid is aligned to the right edge of its parent. |

**Valid values:** `Enum.HorizontalAlignment.Center`, `Enum.HorizontalAlignment.Left`, `Enum.HorizontalAlignment.Right`
