# Enum.Style

The Style Enum is used to set what style of supports the `Class.TrussPart` has.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `AlternatingSupports` | 0 |  | Truss beams contain alternating supports. |
| `BridgeStyleSupports` | 1 |  | Truss beams contain bridge style supports. |
| `NoSupports` | 2 |  | Truss beams do not contain supports. |

**Valid values:** `Enum.Style.AlternatingSupports`, `Enum.Style.BridgeStyleSupports`, `Enum.Style.NoSupports`
