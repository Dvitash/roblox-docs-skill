# Enum.PreferredTextSize

This enum is used with `Class.GuiService.PreferredTextSize` to indicate the player's preferred text size.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `Medium` | 1 |  | The player prefers default text size. |
| `Large` | 2 |  | The player prefers text size slightly increased beyond default. |
| `Larger` | 3 |  | The player prefers text size moderately increased beyond default. |
| `Largest` | 4 |  | The player prefers text size considerably increased beyond default. |

**Valid values:** `Enum.PreferredTextSize.Medium`, `Enum.PreferredTextSize.Large`, `Enum.PreferredTextSize.Larger`, `Enum.PreferredTextSize.Largest`
