# Enum.ApplyStrokeMode

Used by `Class.UIStroke.ApplyStrokeMode` to determine where to apply the stroke.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `Contextual` | 0 |  | The stroke is applied to the parent UI element's text. |
| `Border` | 1 |  | The stroke is applied to the parent UI element's border. |

**Valid values:** `Enum.ApplyStrokeMode.Contextual`, `Enum.ApplyStrokeMode.Border`
