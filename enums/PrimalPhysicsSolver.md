# Enum.PrimalPhysicsSolver

Controls whether the experimental Primal Physics Solver is enabled.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `Default` | 0 |  | Uses the engine-default physics solver state. |
| `Experimental` | 1 |  | The experimental Primal Physics Solver is enabled. |
| `Disabled` | 2 |  | The Primal Physics Solver is disabled. |

**Valid values:** `Enum.PrimalPhysicsSolver.Default`, `Enum.PrimalPhysicsSolver.Experimental`, `Enum.PrimalPhysicsSolver.Disabled`
