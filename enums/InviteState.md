# Enum.InviteState

Describes a player's call invite state.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `Placed` | 0 |  | Call invite was placed. |
| `Accepted` | 1 |  | Call invite was accepted. |
| `Declined` | 2 |  | Call invite was declined. |
| `Missed` | 3 |  | Call invite was missed. |

**Valid values:** `Enum.InviteState.Placed`, `Enum.InviteState.Accepted`, `Enum.InviteState.Declined`, `Enum.InviteState.Missed`
