# Enum.RunContext

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `Legacy` | 0 |  | Runs in legacy script containers dependent on the type of script uses such as `Class.LocalScript` or `Class.Script`. |
| `Server` | 1 |  | Runs on the server. |
| `Client` | 2 |  | Runs on the client. |
| `Plugin` | 3 |  | Runs as a descendant of `Class.Plugin` instances. |

**Valid values:** `Enum.RunContext.Legacy`, `Enum.RunContext.Server`, `Enum.RunContext.Client`, `Enum.RunContext.Plugin`
