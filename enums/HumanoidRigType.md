# Enum.HumanoidRigType

Describes whether or not a character is using the new R15 rig, or the legacy R6 rig.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `R6` | 0 |  | Indicates that a character is using the legacy R6 rig. |
| `R15` | 1 |  | Indicates that a character is using the new R15 rig. |

**Valid values:** `Enum.HumanoidRigType.R6`, `Enum.HumanoidRigType.R15`
