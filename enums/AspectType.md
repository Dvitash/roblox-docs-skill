# Enum.AspectType

Used by `Class.UIAspectRatioConstraint.AspectType` to control sizing behavior.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `FitWithinMaxSize` | 0 |  | The object will be the maximum size possible within its own `Class.GuiBase2d.AbsoluteSize\|AbsoluteSize`. |
| `ScaleWithParentSize` | 1 |  | The object's maximum size will be the size of the parent while still maintaining the aspect ratio. |

**Valid values:** `Enum.AspectType.FitWithinMaxSize`, `Enum.AspectType.ScaleWithParentSize`
