# Enum.VerticalAlignment

Used by `Class.UIGridStyleLayout.VerticalAlignment` to align the layout vertically within its parent.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `Center` | 0 |  | Grid is aligned to the center of its parent along the **Y** axis. |
| `Top` | 1 |  | Grid is aligned to the top edge of its parent. |
| `Bottom` | 2 |  | Grid is aligned to the bottom edge of its parent. |

**Valid values:** `Enum.VerticalAlignment.Center`, `Enum.VerticalAlignment.Top`, `Enum.VerticalAlignment.Bottom`
