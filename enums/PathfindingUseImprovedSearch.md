# Enum.PathfindingUseImprovedSearch

Controls whether `Class.PathfindingService` uses an improved search algorithm.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `Default` | 0 |  | Uses the engine-default pathfinding behavior. |
| `Disabled` | 1 |  | Uses the legacy pathfinding search algorithm. |
| `Enabled` | 2 |  | Uses the improved pathfinding search algorithm. |

**Valid values:** `Enum.PathfindingUseImprovedSearch.Default`, `Enum.PathfindingUseImprovedSearch.Disabled`, `Enum.PathfindingUseImprovedSearch.Enabled`
