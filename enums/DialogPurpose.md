# Enum.DialogPurpose

The DialogPurpose enum is used to set the icon of a `Class.Dialog`.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `Quest` | 0 |  | Shows an exclamation point ('''!'''). |
| `Help` | 1 |  | Shows a question mark ('''?'''). |
| `Shop` | 2 |  | Shows a dollar sign ('''$'''). |

**Valid values:** `Enum.DialogPurpose.Quest`, `Enum.DialogPurpose.Help`, `Enum.DialogPurpose.Shop`
