# Enum.HumanoidCollisionType

The collision `Class.Humanoid` uses.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `OuterBox` | 0 |  | Dynamically sized collision boxes based on mesh sizes. |
| `InnerBox` | 1 |  | Fixed size collision boxes, similar to the classic avatar collision. |

**Valid values:** `Enum.HumanoidCollisionType.OuterBox`, `Enum.HumanoidCollisionType.InnerBox`
