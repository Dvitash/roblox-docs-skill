# Enum.PathWaypointAction

Describes the action to take when a `Datatype.PathWaypoint` is reached.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `Walk` | 0 |  | Walk action needed to reach the next waypoint. |
| `Jump` | 1 |  | Jump action needed to reach the next waypoint. |
| `Custom` | 2 |  |  |

**Valid values:** `Enum.PathWaypointAction.Walk`, `Enum.PathWaypointAction.Jump`, `Enum.PathWaypointAction.Custom`
