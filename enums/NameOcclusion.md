# Enum.NameOcclusion

The NameOcclusion Enum is used to set the `Class.Humanoid.NameOcclusion` property.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `NoOcclusion` | 0 |  | Do not occlude (hide) any humanoid names. |
| `EnemyOcclusion` | 1 |  | Occlude (hide) all enemy humanoid names. |
| `OccludeAll` | 2 |  | Occlude (hide) all humanoid names. |

**Valid values:** `Enum.NameOcclusion.NoOcclusion`, `Enum.NameOcclusion.EnemyOcclusion`, `Enum.NameOcclusion.OccludeAll`
