# Enum.VerticalScrollBarPosition

This enum is used for `Class.ScrollingFrame.VerticalScrollBarPosition` to indicate vertical scroll bar positioning.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `Right` | 0 |  | Right side of the `Class.ScrollingFrame`. |
| `Left` | 1 |  | Left side of the `Class.ScrollingFrame`. |

**Valid values:** `Enum.VerticalScrollBarPosition.Right`, `Enum.VerticalScrollBarPosition.Left`
