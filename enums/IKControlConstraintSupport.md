# Enum.IKControlConstraintSupport

Values for `Class.Workspace.IKControlConstraintSupport`. Sets the support for constraints for `Class.IKControl|IKControls` in your experience.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `Default` | 0 |  | The default option for `Class.IKControl\|IKControls`. The default is currently set to `Enabled`. |
| `Disabled` | 1 |  | Disables support for constraints for `Class.IKControl\|IKControls`. When disabled, `Class.IKControl\|IKControls` still work, but ignore physics constraints. |
| `Enabled` | 2 |  | Enables support for constraints for `Class.IKControl\|IKControls`. |

**Valid values:** `Enum.IKControlConstraintSupport.Default`, `Enum.IKControlConstraintSupport.Disabled`, `Enum.IKControlConstraintSupport.Enabled`
