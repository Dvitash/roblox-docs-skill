# Enum.MaterialPattern

Describes the pattern of material. Affects texture tiling method.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `Regular` | 0 |  | Material has ordered pattern, usually man-made, like brick or wood planks. |
| `Organic` | 1 |  | Material pattern has less repetition, like stone or dirt. |

**Valid values:** `Enum.MaterialPattern.Regular`, `Enum.MaterialPattern.Organic`
