# Enum.FilterResult

Used to determine the result of a text filter request.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `Accepted` | 0 |  | Text accepted by filter. |
| `Rejected` | 1 |  | Text rejected by filter. |

**Valid values:** `Enum.FilterResult.Accepted`, `Enum.FilterResult.Rejected`
