# Enum.CoreGuiType

Represents all available `Class.CoreGui` "types" such as the `Class.Backpack` and text chat window.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `PlayerList` | 0 |  | Dynamically updated `Class.Players` list, commonly used as a [leaderboard](../../../players/leaderboards.md). |
| `Health` | 1 |  | The character's `Class.Humanoid.Health\|Health` bar. |
| `Backpack` | 2 |  | The character's `Class.Backpack` which contains [in‑experience tools](../../../players/tools.md). |
| `Chat` | 3 |  | The [text chat](../../../chat/in-experience-text-chat.md) window. |
| `All` | 4 |  | Refers to all available core UI. |
| `EmotesMenu` | 5 |  | Popup menu of character [emotes](../../../characters/emotes.md). |
| `SelfView` | 6 |  | A player's perspective or view of their own character. |
| `Captures` | 7 |  | A "capture" button along the right side of the screen. |
| `AvatarSwitcher` | 8 |  | The **Avatar Switcher** allows users to change their platform avatar. |
| `ExperienceShop` | 9 |  |  |

**Valid values:** `Enum.CoreGuiType.PlayerList`, `Enum.CoreGuiType.Health`, `Enum.CoreGuiType.Backpack`, `Enum.CoreGuiType.Chat`, `Enum.CoreGuiType.All`, `Enum.CoreGuiType.EmotesMenu`, `Enum.CoreGuiType.SelfView`, `Enum.CoreGuiType.Captures`, `Enum.CoreGuiType.AvatarSwitcher`, `Enum.CoreGuiType.ExperienceShop`
