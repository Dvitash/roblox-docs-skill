# Enum.VirtualCursorMode

Enables Virtual Cursor mode within an experience.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `Default` | 0 |  | The Virtual Cursor is in the default state. |
| `Disabled` | 1 |  | The Virtual Cursor is disabled. |
| `Enabled` | 2 |  | The Virtual Cursor is enabled. |

**Valid values:** `Enum.VirtualCursorMode.Default`, `Enum.VirtualCursorMode.Disabled`, `Enum.VirtualCursorMode.Enabled`
