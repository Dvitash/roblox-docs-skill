# Enum.SaveFilter

Used by `Class.DataModel.SavePlace` to determine the type of save operation This enum determines which aspects of the current place are saved, based on the following options.

**Tags:** Deprecated

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `SaveWorld` | 0 |  | Saves only the world-related data. |
| `SaveGame` | 1 |  | Saves only the game-related data. |
| `SaveAll` | 2 |  | Saves all data, including both world and game-related information. |

**Valid values:** `Enum.SaveFilter.SaveWorld`, `Enum.SaveFilter.SaveGame`, `Enum.SaveFilter.SaveAll`
