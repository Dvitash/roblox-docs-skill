# Enum.MeshType

The MeshType Enum is used to set what type of mesh the `Class.SpecialMesh` is.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `Head` | 0 |  | The mesh is the default `Class.Humanoid` head mesh. |
| `Torso` | 1 |  | The mesh is the default Humanoid torso mesh. |
| `Wedge` | 2 |  | The mesh is a wedge. |
| `Sphere` | 3 |  | The mesh is a sphere. |
| `Cylinder` | 4 |  | The mesh is a cylinder. |
| `FileMesh` | 5 |  | The mesh is determined by the Roblox mesh asset ID provided. |
| `Brick` | 6 |  | The mesh is a brick (just like the shape of a default `Class.BasePart`). |
| `Prism` | 7 | Deprecated | The mesh is a prism. |
| `Pyramid` | 8 | Deprecated | The mesh is a pyramid. |
| `ParallelRamp` | 9 | Deprecated | The mesh is a parallel ramp. |
| `RightAngleRamp` | 10 | Deprecated | The mesh is a right angle ramp. |
| `CornerWedge` | 11 | Deprecated | The mesh is a corner wedge. |

**Valid values:** `Enum.MeshType.Head`, `Enum.MeshType.Torso`, `Enum.MeshType.Wedge`, `Enum.MeshType.Sphere`, `Enum.MeshType.Cylinder`, `Enum.MeshType.FileMesh`, `Enum.MeshType.Brick`, `Enum.MeshType.Prism`, `Enum.MeshType.Pyramid`, `Enum.MeshType.ParallelRamp`, `Enum.MeshType.RightAngleRamp`, `Enum.MeshType.CornerWedge`
