# Enum.VRTouchpad

Used to universally identify a VR touchpad that is used by either the left, or right hand.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `Left` | 0 |  | Left hand touchpad. |
| `Right` | 1 |  | Right hand touchpad. |

**Valid values:** `Enum.VRTouchpad.Left`, `Enum.VRTouchpad.Right`
