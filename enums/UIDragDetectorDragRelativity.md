# Enum.UIDragDetectorDragRelativity

Used with `Class.DragDetector` to set the paradigm which defines the relativity of inputs/outputs from a custom drag function.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `Absolute` | 0 |  | Designates the input and return values as the absolute target position/rotation in the space defined by `Class.UIDragDetector.DragSpace\|DragSpace`. |
| `Relative` | 1 |  | Designates the input and return values as the change from the current position/rotation in the space defined by `Class.UIDragDetector.DragSpace\|DragSpace`. |

**Valid values:** `Enum.UIDragDetectorDragRelativity.Absolute`, `Enum.UIDragDetectorDragRelativity.Relative`
