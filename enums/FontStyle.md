# Enum.FontStyle

Describes whether a `Datatype.Font` style is normal or italic.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `Normal` | 0 |  | The text renders normally with this font. |
| `Italic` | 1 |  | The text renders italic with this font. |

**Valid values:** `Enum.FontStyle.Normal`, `Enum.FontStyle.Italic`
