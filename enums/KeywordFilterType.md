# Enum.KeywordFilterType

Determines if a filter is 'inclusive' or 'exclusive'.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `Include` | 0 |  | Include the matched content. |
| `Exclude` | 1 |  | Exclude the matched content. |

**Valid values:** `Enum.KeywordFilterType.Include`, `Enum.KeywordFilterType.Exclude`
