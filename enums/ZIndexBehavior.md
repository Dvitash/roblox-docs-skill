# Enum.ZIndexBehavior

Used by `Class.LayerCollector.ZIndexBehavior` to control how the `Class.GuiObject.ZIndex` property behaves for descendants.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `Global` | 0 |  | The ZIndex property overrides the default value computed from the depth in the hierarchy. |
| `Sibling` | 1 |  | The ZIndex property controls the order in which the GuiObject renders relative to its siblings. |

**Valid values:** `Enum.ZIndexBehavior.Global`, `Enum.ZIndexBehavior.Sibling`
