# Enum.TextureQueryType

Used to get memory information about textures.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `NonHumanoid` | 0 |  | Texture memory for objects other than `Class.Humanoid`. |
| `NonHumanoidOrphaned` | 1 |  | Unreferenced texture memory for objects other than `Class.Humanoid`. |
| `Humanoid` | 2 |  | Texture memory for `Class.Humanoid`. |
| `HumanoidOrphaned` | 3 |  | Unreferenced texture memory for `Class.Humanoid`. |

**Valid values:** `Enum.TextureQueryType.NonHumanoid`, `Enum.TextureQueryType.NonHumanoidOrphaned`, `Enum.TextureQueryType.Humanoid`, `Enum.TextureQueryType.HumanoidOrphaned`
