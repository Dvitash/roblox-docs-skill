# Enum.OrientationAlignmentMode

The OrientationAlignmentMode Enum is used to select the number of `Class.Attachment|Attachments` used in an alignment.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `OneAttachment` | 0 |  | Use one attachment. |
| `TwoAttachment` | 1 |  | Use two attachments. |

**Valid values:** `Enum.OrientationAlignmentMode.OneAttachment`, `Enum.OrientationAlignmentMode.TwoAttachment`
