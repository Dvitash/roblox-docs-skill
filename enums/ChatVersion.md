# Enum.ChatVersion

Determines whether `Class.TextChatService` should be fully enabled or to allow the legacy system.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `LegacyChatService` | 0 |  | Enables the legacy chat system. |
| `TextChatService` | 1 |  | Enables `Class.TextChatService` chat and prevents legacy chat system behavior. |

**Valid values:** `Enum.ChatVersion.LegacyChatService`, `Enum.ChatVersion.TextChatService`
