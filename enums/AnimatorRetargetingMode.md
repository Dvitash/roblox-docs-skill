# Enum.AnimatorRetargetingMode

Animation Retargeting setting.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `Default` | 0 |  | Retargeting in default state, opt-in initially but subject to change. |
| `Disabled` | 1 |  | Retargeting disabled. |
| `Enabled` | 2 |  | Retargeting enabled. |

**Valid values:** `Enum.AnimatorRetargetingMode.Default`, `Enum.AnimatorRetargetingMode.Disabled`, `Enum.AnimatorRetargetingMode.Enabled`
