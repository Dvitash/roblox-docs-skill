# Enum.UICaptureMode

Used to determine what UI elements should be captured with `Class.CaptureService`.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `All` | 0 |  | Capture all UI elements. |
| `None` | 1 |  | Capture no UI elements. |

**Valid values:** `Enum.UICaptureMode.All`, `Enum.UICaptureMode.None`
