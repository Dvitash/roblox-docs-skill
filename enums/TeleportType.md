# Enum.TeleportType

Determines the type of teleport destination for a `Class.TeleportService` teleport call.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `ToPlace` | 0 |  | The teleport destination is a place (no specific instance). |
| `ToInstance` | 1 |  | The teleport destination is an instance of a place. |
| `ToReservedServer` | 2 |  | The destination of the teleport is a reserved server (a place instance that is reserved for a select group of players). |
| `ToVIPServer` | 3 |  |  |
| `ToInstanceBack` | 4 |  |  |

**Valid values:** `Enum.TeleportType.ToPlace`, `Enum.TeleportType.ToInstance`, `Enum.TeleportType.ToReservedServer`, `Enum.TeleportType.ToVIPServer`, `Enum.TeleportType.ToInstanceBack`
