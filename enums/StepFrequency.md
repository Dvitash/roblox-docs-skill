# Enum.StepFrequency

Enum used with `Class.RunService:BindToSimulation()` to indicate the frequency at which the bound function is called.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `Hz60` | 0 |  | The bound function is called at 60 Hz. |
| `Hz30` | 1 |  | The bound function is called at 30 Hz. |
| `Hz15` | 2 |  | The bound function is called at 15 Hz. |
| `Hz10` | 3 |  | The bound function is called at 10 Hz. |
| `Hz5` | 4 |  | The bound function is called at 5 Hz. |
| `Hz1` | 5 |  | The bound function is called at 1 Hz. |

**Valid values:** `Enum.StepFrequency.Hz60`, `Enum.StepFrequency.Hz30`, `Enum.StepFrequency.Hz15`, `Enum.StepFrequency.Hz10`, `Enum.StepFrequency.Hz5`, `Enum.StepFrequency.Hz1`
