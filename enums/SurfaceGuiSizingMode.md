# Enum.SurfaceGuiSizingMode

Used by `Class.SurfaceGui.SizingMode` to control the sizing behavior of a `Class.SurfaceGui`.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `FixedSize` | 0 |  | Renders with a fixed size set using `Class.SurfaceGui.CanvasSize`. |
| `PixelsPerStud` | 1 |  | Renders with a variable size based on `Class.SurfaceGui.PixelsPerStud` and the SurfaceGui's size in studs. |

**Valid values:** `Enum.SurfaceGuiSizingMode.FixedSize`, `Enum.SurfaceGuiSizingMode.PixelsPerStud`
