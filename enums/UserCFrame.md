# Enum.UserCFrame

Determines what body part is being tracked by a VR device, and what its CFrame actually is.

## Items

| Name | Value | Tags | Summary |
| --- | --- | --- | --- |
| `Head` | 0 |  | The CFrame of the user's head. |
| `LeftHand` | 1 |  | The CFrame of the user's left hand. |
| `RightHand` | 2 |  | The CFrame of the user's right hand. |
| `Floor` | 3 |  |  |

**Valid values:** `Enum.UserCFrame.Head`, `Enum.UserCFrame.LeftHand`, `Enum.UserCFrame.RightHand`, `Enum.UserCFrame.Floor`
