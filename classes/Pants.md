# Pants

**Superclass:** [Clothing](Clothing.md)

Pants is a class in Roblox that displays a pants texture when displayed on a `Class.Humanoid` rig. It must be a sibling of the `Class.Humanoid` and have the appropriate `Class.Pants.PantsTemplate` property set to the desired texture.

## Properties
### `Pants.PantsTemplate`
- **Type:** `ContentId`
- **Summary:** Determines the texture of the `Class.Pants`.
