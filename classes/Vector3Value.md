# Vector3Value

**Superclass:** [ValueBase](ValueBase.md)

A `Vector3Value` class is a container object that holds a single Vector3 value, allowing scripts to communicate and move objects to preset locations.

## Properties
### `Vector3Value.Value`
- **Type:** `Vector3`
- **Summary:** The stored `Datatype.Vector3`.

## Events
### `Vector3Value.Changed(value: Vector3)`
- **Summary:** Fired whenever `Class.Vector3Value.Value` is changed.
