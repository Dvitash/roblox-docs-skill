# ForceField

**Superclass:** [Instance](Instance.md)

A `Class.ForceField` class is used to protect instances from damage and joints when a `Class.SpawnLocation` is reached, and it can be set to be visible for visual effects.

## Properties
### `ForceField.Visible`
- **Type:** `boolean`
- **Summary:** Determines whether or not the `Class.ForceField` particle effect is visible.
