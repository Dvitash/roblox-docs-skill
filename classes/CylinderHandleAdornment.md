# CylinderHandleAdornment

**Superclass:** [HandleAdornment](HandleAdornment.md)

CylinderHandleAdornment is a class for decorating cylinder handles with a specific design, allowing them to listen to input events.

## Properties
### `CylinderHandleAdornment.Angle`
- **Type:** `float`
- **Summary:** Angle of the cylinder handle as in a "pie slice" sector.

### `CylinderHandleAdornment.Height`
- **Type:** `float`
- **Summary:** Height of the cylinder adornment.

### `CylinderHandleAdornment.InnerRadius`
- **Type:** `float`
- **Summary:** Inner radius with which to render a hollow cylinder.

### `CylinderHandleAdornment.Radius`
- **Type:** `float`
- **Summary:** Radius of the cylinder adornment.

### `CylinderHandleAdornment.Shading`
- **Type:** `AdornShading`
