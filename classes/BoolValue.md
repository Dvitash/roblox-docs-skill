# BoolValue

**Superclass:** [ValueBase](ValueBase.md)

A `BoolValue` is a container object for a single boolean value, which is stored in the `Class.BoolValue.Value|Value` property and triggers an event when its value changes.

## Properties
### `BoolValue.Value`
- **Type:** `boolean`
- **Summary:** Used to hold a boolean value.

## Events
### `BoolValue.Changed(value: boolean)`
- **Summary:** Fires whenever the `Class.BoolValue.Value` is changed.

- **Tags:** Deprecated
