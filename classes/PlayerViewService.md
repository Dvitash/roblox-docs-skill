# PlayerViewService

**Superclass:** [Instance](Instance.md)

PlayerViewService is a class that provides methods to retrieve information about a player's view, leveraging the device camera.

## Tags
- NotCreatable
- Service
- NotReplicated

## Methods
### `PlayerViewService:GetDeviceCameraCFrame(player: [Player](Player.md)) -> CFrame`
- **Summary:** Returns a world space `Datatype.CFrame` looking at the player's character.
