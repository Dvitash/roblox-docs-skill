# CanvasGroup

**Superclass:** [GuiObject](GuiObject.md)

CanvasGroup is a class that blends descendants with color/transparency, and it renders descendants as a flattened texture when the ancestor has `Class.LayerCollector` set to `Enum.ZIndexBehavior.Sibling`.

## Properties
### `CanvasGroup.GroupColor3`
- **Type:** `Color3`
- **Summary:** Color tint that applies to all descendants.

### `CanvasGroup.GroupTransparency`
- **Type:** `float`
- **Summary:** Transparency that applies to all descendants.
