# Tween

**Superclass:** [TweenBase](TweenBase.md)

The `Class.Tween` object controls the playback of an interpolation, allowing creation and configuration via a service.

## Properties
### `Tween.Instance`
- **Type:** `[Instance](Instance.md)`
- **Tags:** ReadOnly, NotReplicated
- **Summary:** Read-only property that points to the `Class.Instance` whose properties are being interpolated by the tween.

### `Tween.TweenInfo`
- **Type:** `TweenInfo`
- **Tags:** ReadOnly, NotReplicated
- **Summary:** Read-only property that includes information on how the interpolation of the `Class.Tween` is to be carried out.
