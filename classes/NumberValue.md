# NumberValue

**Superclass:** [ValueBase](ValueBase.md)

A `NumberValue` class represents a single double-precision floating point number, which is stored in the `Class.NumberValue.Value` property and triggers a `Changed` event whenever its value is modified.

## Properties
### `NumberValue.Value`
- **Type:** `double`
- **Summary:** Used to hold a double value.

## Events
### `NumberValue.Changed(value: double)`
- **Summary:** Fires whenever the `Class.NumberValue.Value` is changed.

- **Tags:** Deprecated
