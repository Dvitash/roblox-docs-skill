# SelectionPointLasso

**Superclass:** [SelectionLasso](SelectionLasso.md)

SelectionPointLasso is a 3D GUI object that displays a lasso between a Humanoid and a given Vector3 point. It is deprecated, and developers are advised to use `Class.Beam` instead.

## Tags
- Deprecated

## Properties
### `SelectionPointLasso.Point`
- **Type:** `Vector3`
- **Summary:** Sets the Vector3 target of the lasso object.
