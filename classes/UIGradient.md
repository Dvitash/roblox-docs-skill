# UIGradient

**Superclass:** [UIComponent](UIComponent.md)

Applies a color and transparency gradient to UI elements by configuring the `Class.UIGradient` object with specific `Datatype.ColorSequence`, `Datatype.NumberSequence`, or `Datatype.Vector2` values.

## Properties
### `UIGradient.Color`
- **Type:** `ColorSequence`
- **Summary:** Determines the color blended with the parent GuiObject along the length of the gradient.

### `UIGradient.Enabled`
- **Type:** `boolean`
- **Summary:** Whether the gradient is enabled or not.

### `UIGradient.Offset`
- **Type:** `Vector2`
- **Summary:** Determines the scalar translation of the gradient from the center of the parent GuiObject.

### `UIGradient.Rotation`
- **Type:** `float`
- **Summary:** Determines the clockwise rotation in degrees of the gradient starting from left to right.

### `UIGradient.Transparency`
- **Type:** `NumberSequence`
- **Summary:** Determines how much the parent GuiObject can be seen through along the length of the gradient.
