# DataModelMesh

**Superclass:** [Instance](Instance.md)

The `DataModelMesh` class is an abstract base class that defines the inheritance hierarchy for mesh classes, allowing parented objects to alter their appearance based on a predefined mesh.

## Tags
- NotCreatable
- NotBrowsable

## Properties
### `DataModelMesh.Offset`
- **Type:** `Vector3`
- **Summary:** The Offset of a mesh determines the relative position from the `Class.BasePart.Position` of a `Class.BasePart` that the mesh will be displayed at.

### `DataModelMesh.Scale`
- **Type:** `Vector3`
- **Summary:** The Scale of a mesh determines the size of the mesh relative to its original dimensions.

### `DataModelMesh.VertexColor`
- **Type:** `Vector3`
- **Summary:** Changes the hue of a mesh's texture, used with `Class.FileMesh.TextureId`.
