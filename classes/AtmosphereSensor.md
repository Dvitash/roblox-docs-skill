# AtmosphereSensor

**Superclass:** [SensorBase](SensorBase.md)

This file defines `AtmosphereSensor` as a `Class.SensorBase` class that outputs data on air density and relative wind velocity based on the sensor's position.

## Properties
### `AtmosphereSensor.AirDensity`
- **Type:** `float`
- **Tags:** ReadOnly, NotReplicated
- **Summary:** Measures the density of the air where the sensor is positioned.

### `AtmosphereSensor.RelativeWindVelocity`
- **Type:** `Vector3`
- **Tags:** ReadOnly, NotReplicated
- **Summary:** Reports the wind velocity relative to the sensor.
