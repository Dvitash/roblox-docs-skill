# DataStoreOptions

**Superclass:** [Instance](Instance.md)

DataStoreOptions is a class used to provide additional parameters for `Class.DataStoreService:GetDataStore()`. It contains options for configuring how the `GlobalDataStore` interacts with various scopes.

## Tags
- NotReplicated

## Properties
### `DataStoreOptions.AllScopes`
- **Type:** `boolean`
- **Summary:** Whether the `Class.GlobalDataStore` should work with all scopes.
