# UIGridLayout

**Superclass:** [UIGridStyleLayout](UIGridStyleLayout.md)

Positions sibling UI elements by filling rows using the space of the parent UI element.

## Properties
### `UIGridLayout.AbsoluteCellCount`
- **Type:** `Vector2`
- **Tags:** ReadOnly, NotReplicated
- **Summary:** The number of elements in the grid.

### `UIGridLayout.AbsoluteCellSize`
- **Type:** `Vector2`
- **Tags:** ReadOnly, NotReplicated
- **Summary:** The absolute size of each element in the grid.

### `UIGridLayout.CellPadding`
- **Type:** `UDim2`
- **Summary:** Determines how much space there is between elements in the grid.

### `UIGridLayout.CellSize`
- **Type:** `UDim2`
- **Summary:** Determines the size of each element in the grid.

### `UIGridLayout.FillDirectionMaxCells`
- **Type:** `int`
- **Summary:** Determines the maximum number of cells that may be used in a row or column before the next one is started.

### `UIGridLayout.StartCorner`
- **Type:** `StartCorner`
- **Summary:** Determines from which corner the grid starts laying out UI elements.
