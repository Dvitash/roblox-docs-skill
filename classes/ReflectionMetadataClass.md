# ReflectionMetadataClass

**Superclass:** [ReflectionMetadataItem](ReflectionMetadataItem.md)

Registers information about a class, its members, and provides methods for accessing this data.

## Properties
### `ReflectionMetadataClass.ExplorerImageIndex`
- **Type:** `int`
- **Summary:** A number index, which refers to a specific class icon.

### `ReflectionMetadataClass.ExplorerOrder`
- **Type:** `int`
- **Summary:** Determines how this class is sorted in the Explorer compared to other classes.

### `ReflectionMetadataClass.Insertable`
- **Type:** `boolean`
- **Summary:** Toggles whether or not this object can be inserted through the Advanced Objects menu.

### `ReflectionMetadataClass.PreferredParent`
- **Type:** `string`
- **Summary:** Sets the class that this class would prefer parenting to.
