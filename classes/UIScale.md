# UIScale

**Superclass:** [UIComponent](UIComponent.md)

UIScale is an object that multiplies the size of a parent UI element's `Class.GuiBase2d.AbsoluteSize` by a specific multiplier, defined by the `Scale` property.

## Properties
### `UIScale.Scale`
- **Type:** `float`
- **Summary:** Determines the multiplier to apply to the parent UI element's size.
