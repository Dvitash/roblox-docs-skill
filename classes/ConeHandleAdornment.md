# ConeHandleAdornment

**Superclass:** [HandleAdornment](HandleAdornment.md)

ConeHandleAdornment is a class in the `ConeHandleAdornment` hierarchy that allows an object to be adorned with a cone shape. It can be attached to player parts or the CoreGui, and it listens to input events for dragger tools.

## Properties
### `ConeHandleAdornment.Height`
- **Type:** `float`
- **Summary:** Height of the cone adornment.

### `ConeHandleAdornment.Hollow`
- **Type:** `boolean`

### `ConeHandleAdornment.Radius`
- **Type:** `float`
- **Summary:** Radius of the cone adornment.

### `ConeHandleAdornment.Shading`
- **Type:** `AdornShading`
