# Accessory

**Superclass:** [Accoutrement](Accoutrement.md)

Accessory is a parent class for all accessory instances, which inherit from the Hat system and are cross-compatible with legacy character systems.

## Properties
### `Accessory.AccessoryType`
- **Type:** `AccessoryType`
- **Summary:** Specifies the AccessoryType of the Accessory (eg. Hat, Tshirt, Waist).
