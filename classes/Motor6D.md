# Motor6D

**Superclass:** [Motor](Motor.md)

Motor6D is a class used to create an animatable joint between two `Class.BaseParts` in RX. It allows for the jointing of parts together, which can be referred to as a "rig" for models.

## Properties
### `Motor6D.ChildName`
- **Type:** `string`
- **Tags:** ReadOnly, NotReplicated, NotScriptable

### `Motor6D.ParentName`
- **Type:** `string`
- **Tags:** ReadOnly, NotReplicated, NotScriptable

### `Motor6D.Transform`
- **Type:** `CFrame`
- **Tags:** Hidden, NotReplicated
- **Summary:** Describes the current animation offset of the `Class.Motor6D` joint.
