# Sky

**Superclass:** [Instance](Instance.md)

The `Sky` object changes the default appearance of the experience's sky, allowing users to control the sky's sides and celestial bodies, and enabling features like cubemaps for reflections.

## Properties
### `Sky.CelestialBodiesShown`
- **Type:** `boolean`
- **Summary:** Sets whether the sun, moon, and stars will show.

### `Sky.MoonAngularSize`
- **Type:** `float`
- **Summary:** The perceived angular size of the moon while using this skybox, in degrees.

### `Sky.MoonTextureContent`
- **Type:** `Content`

### `Sky.MoonTextureId`
- **Type:** `ContentId`
- **Summary:** The texture of the moon while using this skybox.

### `Sky.SkyboxBackContent`
- **Type:** `Content`

### `Sky.SkyboxBk`
- **Type:** `ContentId`
- **Summary:** The URL link to a picture for the back surface of the sky.

### `Sky.SkyboxDn`
- **Type:** `ContentId`
- **Summary:** Asset ID for the bottom surface of the skybox.

### `Sky.SkyboxDownContent`
- **Type:** `Content`

### `Sky.SkyboxFrontContent`
- **Type:** `Content`

### `Sky.SkyboxFt`
- **Type:** `ContentId`
- **Summary:** Asset ID for the front surface of the skybox.

### `Sky.SkyboxLeftContent`
- **Type:** `Content`

### `Sky.SkyboxLf`
- **Type:** `ContentId`
- **Summary:** Asset ID for the left surface of the skybox.

### `Sky.SkyboxOrientation`
- **Type:** `Vector3`
- **Summary:** Angle of the skybox, in degrees, with rotation order of **Y**, **X**, **Z**.

### `Sky.SkyboxRightContent`
- **Type:** `Content`

### `Sky.SkyboxRt`
- **Type:** `ContentId`
- **Summary:** Asset ID for the right surface of the skybox.

### `Sky.SkyboxUp`
- **Type:** `ContentId`
- **Summary:** Asset ID for the top surface of the skybox.

### `Sky.SkyboxUpContent`
- **Type:** `Content`

### `Sky.StarCount`
- **Type:** `int`
- **Summary:** How many stars are shown in the skybox.

### `Sky.SunAngularSize`
- **Type:** `float`
- **Summary:** The perceived angular size of the sun while using this skybox, in degrees.

### `Sky.SunTextureContent`
- **Type:** `Content`

### `Sky.SunTextureId`
- **Type:** `ContentId`
- **Summary:** The texture of the sun while using this skybox.
