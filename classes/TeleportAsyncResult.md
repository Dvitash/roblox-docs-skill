# TeleportAsyncResult

**Superclass:** [Instance](Instance.md)

TeleportAsyncResult is an instance returned by `TeleportAsync` that contains the private server ID of the reserved server the players are being teleported to.

## Tags
- NotCreatable

## Properties
### `TeleportAsyncResult.PrivateServerId`
- **Type:** `string`
- **Tags:** ReadOnly, NotReplicated
- **Summary:** The private server ID of the reserved server that the players are being teleported to.

### `TeleportAsyncResult.ReservedServerAccessCode`
- **Type:** `string`
- **Tags:** ReadOnly, NotReplicated
- **Summary:** The access code of the reserved server that the players are being teleported to.
