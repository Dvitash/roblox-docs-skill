# BoxHandleAdornment

**Superclass:** [HandleAdornment](HandleAdornment.md)

BoxHandleAdornment is a class representing a box-shaped handle that can be adorned to a `Class.BasePart`. It allows players to listen to input events for tools like dragger tools if parented to the `Class.PlayerGui` or `Class.CoreGui`.

## Properties
### `BoxHandleAdornment.Shading`
- **Type:** `AdornShading`

### `BoxHandleAdornment.Size`
- **Type:** `Vector3`
- **Summary:** Size of the adornment.
