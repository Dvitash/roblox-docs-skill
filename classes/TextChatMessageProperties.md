# TextChatMessageProperties

**Superclass:** [Instance](Instance.md)

Overrides `Class.TextChatMessage` properties when using callbacks defined in `Class.TextChatService` or `Class.TextChannel.OnIncomingMessage`.

## Properties
### `TextChatMessageProperties.PrefixText`
- **Type:** `string`
- **Summary:** The `Class.TextChatMessage.PrefixText` to override.

### `TextChatMessageProperties.Text`
- **Type:** `string`
- **Summary:** The `Class.TextChatMessage.Text` to override.

### `TextChatMessageProperties.Translation`
- **Type:** `string`
- **Summary:** The `Class.TextChatMessage.Translation` to override.
