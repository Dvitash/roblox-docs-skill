# FileMesh

**Superclass:** [DataModelMesh](DataModelMesh.md)

FileMesh is a class used to apply a mesh to a `Class.BasePart` when parented to it, inheriting properties from `Class.SpecialMesh`.

## Properties
### `FileMesh.MeshId`
- **Type:** `ContentId`
- **Summary:** The MeshId is the content ID of the mesh that is to be displayed.

### `FileMesh.TextureId`
- **Type:** `ContentId`
- **Summary:** The TextureId is the content ID of the texture that is to be applied to the mesh.
