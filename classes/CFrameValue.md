# CFrameValue

**Superclass:** [ValueBase](ValueBase.md)

A `CFrameValue` class is a container object for a single `Datatype.CFrame` value, providing methods to track changes and events.

## Properties
### `CFrameValue.Value`
- **Type:** `CFrame`
- **Summary:** Used to hold a `Datatype.CFrame` value.

## Events
### `CFrameValue.Changed(value: CFrame)`
- **Summary:** Fired whenever the `Class.CFrameValue.Value` of the CFrameValue is changed.

- **Tags:** Deprecated
