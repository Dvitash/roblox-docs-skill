# FloorWire

**Superclass:** [GuiBase3d](GuiBase3d.md)

The `FloorWire` class is a class for creating wires from two properties, `Class.FloorWire.From` and `Class.FloorWire.To`, which must be set to `Class.BasePart`.

## Tags
- Deprecated

## Properties
### `FloorWire.CycleOffset`
- **Type:** `float`
- **Summary:** A decimal number between 0 and 1, through which you can control how far all of the decals are along the wire.

### `FloorWire.From`
- **Type:** `[BasePart](BasePart.md)`
- **Summary:** The object that the FloorWire travels from.

### `FloorWire.StudsBetweenTextures`
- **Type:** `float`
- **Summary:** The number of studs between each FloorWire segment.

### `FloorWire.Texture`
- **Type:** `ContentId`
- **Summary:** Sets the texture to be displayed on the FloorWire.

### `FloorWire.TextureSize`
- **Type:** `Vector2`
- **Summary:** Sets the size of the texture used with the FloorWire.

### `FloorWire.To`
- **Type:** `[BasePart](BasePart.md)`
- **Summary:** The object that the FloorWire travels to.

### `FloorWire.Velocity`
- **Type:** `float`
- **Summary:** The speed that the textures flow along the wire.

### `FloorWire.WireRadius`
- **Type:** `float`
- **Summary:** The radius of the wire.
