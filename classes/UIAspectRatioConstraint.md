# UIAspectRatioConstraint

**Superclass:** [UIConstraint](UIConstraint.md)

The `UIAspectRatioConstraint` class enforces a specific aspect ratio on a parent element, ensuring the width-to-height dimensions of child elements maintain a consistent ratio.

## Properties
### `UIAspectRatioConstraint.AspectRatio`
- **Type:** `float`
- **Summary:** Determines the width-to-height ratio to maintain.

### `UIAspectRatioConstraint.AspectType`
- **Type:** `AspectType`
- **Summary:** Determines how the maximum size of the object is limited.

### `UIAspectRatioConstraint.DominantAxis`
- **Type:** `DominantAxis`
- **Summary:** Determines the axis to use when setting the new size of the object.
