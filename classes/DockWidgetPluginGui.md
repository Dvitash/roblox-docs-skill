# DockWidgetPluginGui

**Superclass:** [PluginGui](PluginGui.md)

DockWidgetPluginGui is a class used to create dockable Roblox Studio windows that display widgets similar to the built-in Animation Editor or Terrain Tools.

## Tags
- NotCreatable
- NotReplicated

## Properties
### `DockWidgetPluginGui.HostWidgetWasRestored`
- **Type:** `boolean`
- **Tags:** ReadOnly, NotReplicated
- **Summary:** Describes whether the previous state of this DockWidgetPluginGui was restored when it was created.
