# Clothing

**Superclass:** [CharacterAppearance](CharacterAppearance.md)

Clothing is the base class for clothing objects, defining how `Class.Shirt` and `Class.Pants` objects are colored.

## Tags
- NotCreatable

## Properties
### `Clothing.Color3`
- **Type:** `Color3`
- **Summary:** Determines the colorization to be applied to the `Class.Clothing` texture.
