# BackpackItem

**Superclass:** [Model](Model.md)

BackpackItem is an abstract class for backpack items, including HopperBins and Tools. It provides properties for displaying the tool's texture content and icon.

## Tags
- NotCreatable

## Properties
### `BackpackItem.TextureContent`
- **Type:** `Content`
- **Summary:** The texture icon that is displayed for a tool in the player's backpack. Only supports asset URIs.

### `BackpackItem.TextureId`
- **Type:** `ContentId`
- **Summary:** The texture icon that is displayed for a tool in the player's backpack.
