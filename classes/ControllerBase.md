# ControllerBase

**Superclass:** [Instance](Instance.md)

ControllerBase is a class representing a controller base, which includes methods for controlling movement and balance. It inherits from several other controllers and has specific properties and methods related to movement and simulation.

## Tags
- NotCreatable

## Properties

- **Type:** `boolean`
- **Tags:** ReadOnly, NotReplicated

- **Type:** `boolean`

### `ControllerBase.MoveSpeedFactor`
- **Type:** `float`
- **Summary:** The value multiplied by the `Class.ControllerManager.BaseMoveSpeed`.
