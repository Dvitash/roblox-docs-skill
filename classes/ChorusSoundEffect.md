# ChorusSoundEffect

**Superclass:** [SoundEffect](SoundEffect.md)

ChorusSoundEffect is a class that simulates multiple voices or instruments playing the same part by overlaying copies of the original sound, resulting in a chorus effect.

## Properties
### `ChorusSoundEffect.Depth`
- **Type:** `float`
- **Summary:** Controls how intense the effect is.

### `ChorusSoundEffect.Mix`
- **Type:** `float`
- **Summary:** Percentage of the original sound that will be applied to the filter.

### `ChorusSoundEffect.Rate`
- **Type:** `float`
- **Summary:** How frequently the pitch variation changes.
