# SpecialMesh

**Superclass:** [FileMesh](FileMesh.md)

The `SpecialMesh` class applies a mesh to a `Class.BasePart` based on a specific `Class.SpecialMesh.MeshType`. It supports various shapes and allows for scaling, but does not affect collision modeling.

## Properties
### `SpecialMesh.MeshType`
- **Type:** `MeshType`
- **Summary:** Determines the type of mesh that will be applied to the `Class.BasePart` the `Class.SpecialMesh` is parented to.
