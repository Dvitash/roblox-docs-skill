# NetworkReplicator

**Superclass:** [Instance](Instance.md)

NetworkReplicator is an object that handles the replication of objects from the server to the client or vice versa.

## Tags
- NotCreatable
- NotReplicated

## Methods
### `NetworkReplicator:GetPlayer() -> [Instance](Instance.md)`
- **Summary:** Returns the player that is connected to the NetworkReplicator.
