# UITextSizeConstraint

**Superclass:** [UIConstraint](UIConstraint.md)

UITextSizeConstraint is a constraint class that ensures the size of text rendered by `Class.GuiObject` classes within a specified range.

## Properties
### `UITextSizeConstraint.MaxTextSize`
- **Type:** `int`
- **Summary:** The largest size in pixels the font is allowed to be.

### `UITextSizeConstraint.MinTextSize`
- **Type:** `int`
- **Summary:** The smallest size in pixels the font is allowed to be.
