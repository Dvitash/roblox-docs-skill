# CoreGui

**Superclass:** [BasePlayerGui](BasePlayerGui.md)

The CoreGui is a service used by Roblox to manage game elements, and it can be configured through methods in `Class.StarterGui` or `Class.PlayerGui`.

## Tags
- NotCreatable
- Service
- NotReplicated

## Properties
### `CoreGui.SelectionImageObject`
- **Type:** `[GuiObject](GuiObject.md)`

### `CoreGui.Version`
- **Type:** `int`
- **Tags:** ReadOnly, NotReplicated
- **Summary:** The current version of the CoreGui. Every time the CoreGui is majorly changed, this number is increased.
