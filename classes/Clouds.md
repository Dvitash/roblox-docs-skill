# Clouds

**Superclass:** [Instance](Instance.md)

Renders realistic clouds that drift across the sky, allowing users to adjust cloud cover, density, and color.

## Properties
### `Clouds.Color`
- **Type:** `Color3`
- **Summary:** Controls the material color of cloud particles.

### `Clouds.Cover`
- **Type:** `float`
- **Summary:** Defines the cloud cover within the overall skyscape layer.

### `Clouds.Density`
- **Type:** `float`
- **Summary:** Controls the particulate density of clouds.

### `Clouds.Enabled`
- **Type:** `boolean`
- **Summary:** Toggles rendering of the `Class.Clouds` object.
