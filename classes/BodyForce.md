# BodyForce

**Superclass:** [BodyMover](BodyMover.md)

BodyForce is a class that applies a constant force to an object, affecting its acceleration. It is deprecated and should be replaced with `Class.VectorForce`.

## Tags
- Deprecated

## Properties
### `BodyForce.Force`
- **Type:** `Vector3`
- **Summary:** Determines the force exerted on each axis.

### `BodyForce.force`
- **Type:** `Vector3`
- **Tags:** NotReplicated, Deprecated
- **Summary:** Determines the amount of force applied on each axis.
