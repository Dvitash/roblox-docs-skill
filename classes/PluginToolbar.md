# PluginToolbar

**Superclass:** [Instance](Instance.md)

A `PluginToolbar` object is a container for `Class.PluginToolbarButton` instances, allowing users to initiate single-off actions in Studio.

## Tags
- NotCreatable

## Methods
### `PluginToolbar:CreateButton(buttonId: string, tooltip: string, iconname: string, text: string) -> [PluginToolbarButton](PluginToolbarButton.md)`
- **Summary:** Creates a `Class.PluginToolbarButton` that allows the user to initiate a single, one-off action in Studio.
