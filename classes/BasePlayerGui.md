# BasePlayerGui

**Superclass:** [Instance](Instance.md)

BasePlayerGui is an abstract class that all GUI drawing storage classes inherit from, providing a base for storing and managing GUI objects.

## Tags
- NotCreatable

## Methods
### `BasePlayerGui:GetGuiObjectsAtPosition(x: int, y: int) -> Instances`
- **Summary:** Returns a list of all `Class.GuiObject` instances occupying the given point on the screen.
