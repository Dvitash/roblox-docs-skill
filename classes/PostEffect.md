# PostEffect

**Superclass:** [Instance](Instance.md)

PostEffect is an abstract base class for post-processing effects, which change how objects look after rendering. It is used to manage effects like `Class.BloomEffect` and `Class.ColorCorrectionEffect`.

## Tags
- NotCreatable

## Properties
### `PostEffect.Enabled`
- **Type:** `boolean`
- **Summary:** Toggles whether or not the PostEffect is enabled.
