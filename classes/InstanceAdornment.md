# InstanceAdornment

**Superclass:** [GuiBase3d](GuiBase3d.md)

This file defines the `InstanceAdornment` class, which is a base class for objects that adorn `Class.Instance` classes. It determines which `Class.Instance` to render in 3D relative to the `adornee`.

## Tags
- NotCreatable

## Properties
### `InstanceAdornment.Adornee`
- **Type:** `[Instance](Instance.md)`
- **Summary:** Which `Class.Instance` to adorn.
