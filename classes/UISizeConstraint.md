# UISizeConstraint

**Superclass:** [UIConstraint](UIConstraint.md)

UISizeConstraint ensures that an object's size does not exceed the maximum or minimum sizes defined by `Class.UISizeConstraint.MaxSize` and `Class.UISizeConstraint.MinSize`.

## Properties
### `UISizeConstraint.MaxSize`
- **Type:** `Vector2`
- **Summary:** The largest size, in pixels, the parent object is allowed to be.

### `UISizeConstraint.MinSize`
- **Type:** `Vector2`
- **Summary:** The smallest size, in pixels, the object is allowed to be.
