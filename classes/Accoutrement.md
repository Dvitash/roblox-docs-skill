# Accoutrement

**Superclass:** [Instance](Instance.md)

An Accoutrement is an object that attaches to a player's character, allowing for changes in position and rotation.

## Properties
### `Accoutrement.AttachmentForward`
- **Type:** `Vector3`
- **Tags:** Hidden, NotReplicated
- **Summary:** Sets the offset position of the object on the Player.

### `Accoutrement.AttachmentPoint`
- **Type:** `CFrame`
- **Summary:** The exact CFrame of the Accoutrement.

### `Accoutrement.AttachmentPos`
- **Type:** `Vector3`
- **Tags:** Hidden, NotReplicated
- **Summary:** Sets the position of the object on the Player.

### `Accoutrement.AttachmentRight`
- **Type:** `Vector3`
- **Tags:** Hidden, NotReplicated
- **Summary:** Sets the offset position of the object on the Player.

### `Accoutrement.AttachmentUp`
- **Type:** `Vector3`
- **Tags:** Hidden, NotReplicated
- **Summary:** Sets the offset position of the object on the Player.
