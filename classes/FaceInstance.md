# FaceInstance

**Superclass:** [Instance](Instance.md)

FaceInstance is an abstract class that inherits from Decal and Texture, providing methods for setting the face of a parent `BasePart`.

## Tags
- NotCreatable
- NotBrowsable

## Properties
### `FaceInstance.Face`
- **Type:** `NormalId`
- **Summary:** Sets which face of the parent `Class.BasePart` the object appears on.
