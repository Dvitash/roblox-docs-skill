# VideoService

**Superclass:** [Instance](Instance.md)

An internal service that provides no functionality to developers, it is marked as NotCreatable and has specific documentation for creating a `Class.VideoSampler`.

## Tags
- NotCreatable
- Service

## Methods
### `VideoService:CreateVideoSamplerAsync(content: Content, options: Dictionary?) -> [VideoSampler](VideoSampler.md)`
- **Tags:** Yields
- **Summary:** Creates a `Class.VideoSampler` that samples frames from the provided video content.
