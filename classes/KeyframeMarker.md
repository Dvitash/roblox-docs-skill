# KeyframeMarker

**Superclass:** [Instance](Instance.md)

KeyframeMarker is an instance used to represent an event that will occur when a `Class.Keyframe` is hit, allowing for attaching and listening to specific markers.

## Properties
### `KeyframeMarker.Value`
- **Type:** `string`
- **Summary:** A value that is specified for a `Class.KeyframeMarker`.
