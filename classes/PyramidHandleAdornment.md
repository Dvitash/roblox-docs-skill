# PyramidHandleAdornment

**Superclass:** [HandleAdornment](HandleAdornment.md)

PyramidHandleAdornment is a class in the `pyramid` module that allows for decorating pyramid-shaped handles on `Class.BasePart` by attaching to player or core gui elements.

## Properties
### `PyramidHandleAdornment.Height`
- **Type:** `float`
- **Summary:** Height of the pyramid adornment.

### `PyramidHandleAdornment.Shading`
- **Type:** `AdornShading`

### `PyramidHandleAdornment.Sides`
- **Type:** `int`
- **Summary:** Number of sides for the pyramid adornment.

### `PyramidHandleAdornment.Size`
- **Type:** `float`
- **Summary:** Size of the pyramid adornment's base.
