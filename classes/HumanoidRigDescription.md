# HumanoidRigDescription

**Superclass:** [Instance](Instance.md)

### HumanoidRigDescription.Chest - Stores the joint mapping, reference T-pose, and per-joint properties for a bipedal `Humanoid` character rig. - Each joint can reference an `Instance` (e.g., `Motor6D`, `AnimationConstraint`) to define T-pose adjustments, rotation ranges, and size for visualization. - The `AutoRig` method automatically populates joint references from a model or assigns joints individually. - Supports both R6 (6-joint) and R15 (15-joint) rig subsets and more than 22 joints. ###…
