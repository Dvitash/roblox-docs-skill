# ShirtGraphic

**Superclass:** [CharacterAppearance](CharacterAppearance.md)

ShirtGraphic is a class used to apply a texture to the front surface of a character's torso, which displays t-shirts.

## Properties
### `ShirtGraphic.Color3`
- **Type:** `Color3`
- **Summary:** Determines the colorization to be applied to the `Class.ShirtGraphic` texture.

### `ShirtGraphic.Graphic`
- **Type:** `ContentId`
- **Summary:** The content ID link pointing to the `Class.ShirtGraphic` texture hosted on the Roblox website. This property sets the texture associated with a t-shirt.
