# ChatWindowMessageProperties

**Superclass:** [TextChatMessageProperties](TextChatMessageProperties.md)

ChatWindowMessageProperties is a class used to customize the appearance of text chat messages sent to the ChatWindow. It inherits from `Class.TextChatMessageProperties` and allows for customizing font, prefix text, and text color/stroke transparency.

## Tags
- NotCreatable

## Properties
### `ChatWindowMessageProperties.FontFace`
- **Type:** `Datatype.Font`
- **Summary:** Font used to render text in the chat window.

### `ChatWindowMessageProperties.PrefixTextProperties`
- **Type:** `[ChatWindowMessageProperties](ChatWindowMessageProperties.md)`
- **Summary:** Determines the properties of the `Class.TextChatMessage.PrefixText|PrefixText` preceding the chat message.

### `ChatWindowMessageProperties.TextColor3`
- **Type:** `Color3`
- **Summary:** Color of the text in the chat window.

### `ChatWindowMessageProperties.TextSize`
- **Type:** `int`
- **Summary:** Size of the text in the chat window.

### `ChatWindowMessageProperties.TextStrokeColor3`
- **Type:** `Color3`
- **Summary:** Stroke color applied to text in the chat window.

### `ChatWindowMessageProperties.TextStrokeTransparency`
- **Type:** `double`
- **Summary:** Transparency of the stroke applied to text in the chat window.
