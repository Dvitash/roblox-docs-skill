# GeneratedFolder

**Superclass:** [Folder](Folder.md)

GeneratedFolder is a container used to store deterministic branches of `Class.DataModel` generation results, which are used by `Class.ProceduralModel` to organize generated content.

## Methods
### `GeneratedFolder:SetPrimaryPart(part: [BasePart](BasePart.md)) -> ()`
- **Summary:** Specifies which part within the `GeneratedFolder` should be set as the `Class.Model.PrimaryPart|PrimaryPart` of the `Class.ProceduralModel`.
