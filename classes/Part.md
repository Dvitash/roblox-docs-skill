# Part

**Superclass:** [FormFactorPart](FormFactorPart.md)

A `Part` class represents a common shape type found in `Class.BaseParts`. It has five primitive shapes and is used for creating custom part shapes using solid modeling or collision detection.

## Properties
### `Part.Shape`
- **Type:** `PartType`
- **Tags:** NotReplicated
- **Summary:** Sets the overall shape of the object.
