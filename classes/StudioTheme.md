# StudioTheme

**Superclass:** [Instance](Instance.md)

StudioTheme is a class representing the Studio Theme, which provides methods to retrieve the current theme color based on specific style and modifier settings.

## Tags
- NotCreatable
- NotReplicated

## Methods
### `StudioTheme:GetColor(styleguideitem: StudioStyleGuideColor, modifier: StudioStyleGuideModifier) -> Color3`
- **Summary:** Returns the color corresponding to the arguments provided.
