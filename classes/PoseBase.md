# PoseBase

**Superclass:** [Instance](Instance.md)

PoseBase is the base class for all 'Pose Instance' objects, defining common properties and methods for easing and animating poses.

## Tags
- NotCreatable

## Properties
### `PoseBase.EasingDirection`
- **Type:** `PoseEasingDirection`
- **Summary:** The easing direction to use to reach the next Pose's value.

### `PoseBase.EasingStyle`
- **Type:** `PoseEasingStyle`
- **Summary:** The easing style to use to reach the next Pose's value.

### `PoseBase.Weight`
- **Type:** `float`
