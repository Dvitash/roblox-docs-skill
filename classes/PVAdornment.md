# PVAdornment

**Superclass:** [GuiBase3d](GuiBase3d.md)

An abstract class `PVAdornment` allows inheritors to be adorned to objects of the `Class.PVInstance` class, enabling attachment and positioning of adornments relative to their parent object.

## Tags
- NotCreatable

## Properties
### `PVAdornment.Adornee`
- **Type:** `[PVInstance](PVInstance.md)`
- **Summary:** The `Class.PVInstance` which this `Class.PVAdornment` is attached to.
