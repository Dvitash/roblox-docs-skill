# LineHandleAdornment

**Superclass:** [HandleAdornment](HandleAdornment.md)

LineHandleAdornment is a class for decorating line handles on `Class.BasePart` by positioning them along the `CFrame`. It allows handles to listen to input events, such as dragger tools, and can be parented to player GUIs or CoreGUI.

## Properties
### `LineHandleAdornment.Length`
- **Type:** `float`
- **Summary:** Length of the line.

### `LineHandleAdornment.Thickness`
- **Type:** `float`
- **Summary:** Thickness of the line in pixels.
