# GlobalSettings

**Superclass:** [GenericSettings](GenericSettings.md)

GlobalSettings is a base class for Roblox Studio settings, which can be accessed via the `settings()` function. It contains various classes for managing different types of Roblox Studio settings.

## Tags
- NotCreatable
- NotBrowsable

## Methods
### `GlobalSettings:GetFFlag(name: string) -> boolean`
- **Summary:** Returns the value of an FFlag if it exists.

### `GlobalSettings:GetFVariable(name: string) -> string`
- **Summary:** Returns the value of an FVariable, if it exists.
