# UIShadow

**Superclass:** [UIComponent](UIComponent.md)

Renders a shadow below a parent UI instance, allowing for adjustable blurriness, color/transparency changes, and position/size adjustments.

## Tags
- NotBrowsable

## Properties
### `UIShadow.BlurRadius`
- **Type:** `UDim`
- **Summary:** Determines the shadow's blurriness.

### `UIShadow.Color`
- **Type:** `Color3`
- **Summary:** Determines the shadow's color.

### `UIShadow.Offset`
- **Type:** `UDim2`
- **Summary:** Moves the shadow relative to the parent's position.

### `UIShadow.Spread`
- **Type:** `UDim2`
- **Summary:** Expands or shrinks the shadow relative to the parent's size.

### `UIShadow.Transparency`
- **Type:** `float`
- **Summary:** Sets the shadow's transparency.

### `UIShadow.ZIndex`
- **Type:** `int`
- **Summary:** Determines the shadow's render order relative to sibling `UIShadow` instances.

### `UIShadow.Enabled`
- **Type:** `boolean`
- **Summary:** Determines whether the `UIShadow` is visible.
