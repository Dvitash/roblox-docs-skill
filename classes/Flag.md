# Flag

**Superclass:** [Tool](Tool.md)

The `Flag` class is a unit spawned with `Class.FlagStand` objects, which respawns when captured and allows players to carry flags.

## Tags
- Deprecated

## Properties
### `Flag.TeamColor`
- **Type:** `BrickColor`
- **Summary:** The `Class.Team` this flag is for. Corresponds with the TeamColors in the `Class.Teams` service.
