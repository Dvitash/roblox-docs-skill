# PluginMouse

**Superclass:** [Mouse](Mouse.md)

PluginMouse is a class in `Class.Plugin` that provides `Class.Plugin|Plugins` access to the mouse, similar to `Class.Mouse`. It includes functions for tracking selection while dragging.

## Tags
- NotCreatable

## Events
### `PluginMouse.DragEnter(instances: Instances)`
- **Summary:** Fired when Instances are being selected while the mouse is dragging.
