# WrapTarget

**Superclass:** [BaseWrap](BaseWrap.md)

WrapTarget is a class defining a 3D body with only an outer surface or Outer Cage, used for applying 3D accessories.

## Properties
### `WrapTarget.Color`
- **Type:** `Color3`
- **Tags:** NotReplicated, NotScriptable
- **Summary:** Sets color used for the debug rendering. See `Class.WrapTarget.DebugMode`.

### `WrapTarget.DebugMode`
- **Type:** `WrapTargetDebugMode`
- **Tags:** NotReplicated, NotScriptable
- **Summary:** Allows switching between different debugging visualization modes for cage meshes.

### `WrapTarget.Stiffness`
- **Type:** `float`
- **Summary:** Defines how much the body mesh can be compressed by clothing.
