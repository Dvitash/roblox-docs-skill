# ScreenshotHud

**Superclass:** [Instance](Instance.md)

The `ScreenshotHud` is a 2D user interface that allows users to capture and save screenshots of their device. It includes UI elements for capturing the experience name and Roblox branding, and properties for controlling visibility and accessibility.

## Tags
- NotCreatable
- NotReplicated

## Properties
### `ScreenshotHud.CameraButtonIcon`
- **Type:** `ContentId`
- **Summary:** Asset ID of the icon used for the camera button.

### `ScreenshotHud.CameraButtonPosition`
- **Type:** `UDim2`
- **Summary:** Screen location of the camera button.

### `ScreenshotHud.CloseButtonPosition`
- **Type:** `UDim2`
- **Summary:** Screen location of the close button.

### `ScreenshotHud.CloseWhenScreenshotTaken`
- **Type:** `boolean`
- **Summary:** Whether the `Class.ScreenshotHud` closes automatically when a screenshot is taken.

### `ScreenshotHud.ExperienceNameOverlayEnabled`
- **Type:** `boolean`
- **Tags:** Hidden, Deprecated
- **Summary:** Whether the experience name in the overlay is enabled.

### `ScreenshotHud.HideCoreGuiForCaptures`
- **Type:** `boolean`

### `ScreenshotHud.HidePlayerGuiForCaptures`
- **Type:** `boolean`

### `ScreenshotHud.OverlayFont`
- **Type:** `Enum.Font`
- **Tags:** Hidden, Deprecated
- **Summary:** `Datatype.Font` used for the experience name in the overlay.

### `ScreenshotHud.UsernameOverlayEnabled`
- **Type:** `boolean`
- **Tags:** Hidden, Deprecated
- **Summary:** Currently disabled with no effect.

### `ScreenshotHud.Visible`
- **Type:** `boolean`
- **Summary:** Determines whether the `Class.ScreenshotHud` is visible.
