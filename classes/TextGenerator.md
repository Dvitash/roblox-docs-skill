# TextGenerator

**Superclass:** [Instance](Instance.md)

Gives access to a large language model (LLM) for text generation, allowing users to create interactive NPCs by providing system and user prompts.

## Properties
### `TextGenerator.Seed`
- **Type:** `int`
- **Summary:** Sets a fixed seed for the random number generator, allowing reproducible responses in cases where the same input parameters are used across multiple requests.

### `TextGenerator.SystemPrompt`
- **Type:** `string`
- **Summary:** Provides context to the model about its role, tone, or behavior during conversation.

### `TextGenerator.Temperature`
- **Type:** `float`
- **Summary:** Controls the "creativity" or randomness of the model's responses.

### `TextGenerator.TopP`
- **Type:** `float`
- **Summary:** Helps the AI model narrow or expand the range of possible words to sample from while generating the next token.

## Methods
### `TextGenerator:GenerateTextAsync(request: Dictionary) -> Dictionary`
- **Tags:** Yields
- **Summary:** Returns text generated by an LLM based on the provided system and user prompts.
