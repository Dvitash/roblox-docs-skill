# GenerationService

**Superclass:** [Instance](Instance.md)

GenerationService is a class for generating 3D objects from text prompts using Roblox's Cube 3D foundation model. It allows developers to generate various models, including environmental props and vehicles, with optional intermediate results.

## Tags
- NotCreatable
- Service

## Methods
### `GenerationService:GenerateMeshAsync(inputs: Dictionary, player: [Player](Player.md), options: Dictionary, intermediateResultCallback: Function?) -> Tuple`
- **Tags:** Yields
- **Summary:** Starts the generation of a new 3D mesh from a text prompt and returns unique IDs used to track and retrieve the result.

### `GenerationService:GenerateModelAsync(inputs: Dictionary, schema: Dictionary, options: Dictionary?) -> Tuple`
- **Tags:** Yields
- **Summary:** Enables generation of multi-mesh geometries according to provided `inputs` and a `schema`.

### `GenerationService:LoadGeneratedMeshAsync(generationId: string) -> [MeshPart](MeshPart.md)`
- **Tags:** Yields
- **Summary:** Retrieves and loads a mesh generated by `Class.GenerationService:GenerateMeshAsync()` using the provided `generationId`.
