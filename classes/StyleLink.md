# StyleLink

**Superclass:** [Instance](Instance.md)

This file defines a `StyleLink` class that links a `Class.StyleSheet` to the instance tree of an object, allowing the referenced style to be applied to all descendants.

## Properties
### `StyleLink.StyleSheet`
- **Type:** `[StyleSheet](StyleSheet.md)`
- **Summary:** The `Class.StyleSheet` to link to the parent such that the parent's descendants are styled accordingly.
