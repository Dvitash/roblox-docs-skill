# ColorGradingEffect

**Superclass:** [PostEffect](PostEffect.md)

ColorGradingEffect is a class that modifies how color values are converted to the screen's color range, affecting the mood and appearance of a place.

## Properties
### `ColorGradingEffect.TonemapperPreset`
- **Type:** `TonemapperPreset`
- **Summary:** Specifies which tone mapper preset to use.
