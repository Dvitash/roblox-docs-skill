# NumberPose

**Superclass:** [PoseBase](PoseBase.md)

The `NumberPose` class is a building block for facial animation, holding the value applied to a specific FACS control. It acts as an abstraction layer between muscle movements and defined facial rigs.

## Properties
### `NumberPose.Value`
- **Type:** `double`
- **Summary:** The value that will be applied to the FACS control corresponding to the `Class.NumberPose`.
