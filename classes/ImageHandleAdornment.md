# ImageHandleAdornment

**Superclass:** [HandleAdornment](HandleAdornment.md)

ImageHandleAdornment is an image handle that can be adorned to a `Class.BasePart`. It allows handles to listen to input events, such as making dragger tools, when parented to player components like `Class.PlayerGui` or `Class.CoreGui`.

## Properties
### `ImageHandleAdornment.Image`
- **Type:** `ContentId`
- **Summary:** Image to draw for the adornment.

### `ImageHandleAdornment.Size`
- **Type:** `Vector2`
- **Summary:** Size of the image in studs.
