# Sparkles

**Superclass:** [Instance](Instance.md)

Sparkles is a particle emitter that emits particles when parented to `Class.BasePart` or `Class.Attachment`. It has specific rules for rendering and handling, such as stopping emission when the parent object is destroyed.

## Properties
### `Sparkles.Color`
- **Type:** `Color3`
- **Tags:** Hidden, NotReplicated
- **Summary:** Determines the color of the sparkle particles.

### `Sparkles.Enabled`
- **Type:** `boolean`
- **Summary:** Determines whether sparkles are emit.

### `Sparkles.LocalTransparencyModifier`
- **Type:** `float`
- **Tags:** Hidden, NotReplicated

### `Sparkles.SparkleColor`
- **Type:** `Color3`
- **Summary:** Determines the color of the sparkle particles.
