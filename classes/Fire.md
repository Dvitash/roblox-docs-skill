# Fire

**Superclass:** [Instance](Instance.md)

The `Class.Fire` class is a preconfigured particle emitter with visual aesthetics similar to fire, which can be used for quick effects or detailed work.

## Properties
### `Fire.Color`
- **Type:** `Color3`
- **Summary:** Determines the color of the primary (outer) flame particles.

### `Fire.Enabled`
- **Type:** `boolean`
- **Summary:** Determines whether flame particles are emit.

### `Fire.Heat`
- **Type:** `float`
- **Tags:** NotReplicated
- **Summary:** Determines the velocity at which particles are emit.

### `Fire.LocalTransparencyModifier`
- **Type:** `float`
- **Tags:** Hidden, NotReplicated
- **Summary:** A multiplier for the `Class.Fire` object's transparency that is only visible to the local client.

### `Fire.SecondaryColor`
- **Type:** `Color3`
- **Summary:** Determines the color of the of the secondary (inner) flame particles.

### `Fire.Size`
- **Type:** `float`
- **Tags:** NotReplicated
- **Summary:** Determines the size of the flame particles.

### `Fire.size`
- **Type:** `float`
- **Tags:** NotReplicated, Deprecated

### `Fire.TimeScale`
- **Type:** `float`
- **Summary:** Controls the speed of the particle effect.
