# DialogChoice

**Superclass:** [Instance](Instance.md)

DialogChoice is a class used to craft further choices available when a player initiates a dialog conversation with an NPC.

## Properties
### `DialogChoice.GoodbyeChoiceActive`
- **Type:** `boolean`
- **Summary:** Toggles whether the goodbye option will be displayed.

### `DialogChoice.GoodbyeDialog`
- **Type:** `string`
- **Summary:** Sets the sentence that the dialog will show to the player when the chat ends.

### `DialogChoice.ResponseDialog`
- **Type:** `string`
- **Summary:** Sets what the NPC will say when the player chooses this DialogChoice.

### `DialogChoice.UserDialog`
- **Type:** `string`
- **Summary:** Sets what the player will say when they choose this DialogChoice.
