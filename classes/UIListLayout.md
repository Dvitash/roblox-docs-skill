# UIListLayout

**Superclass:** [UIGridStyleLayout](UIGridStyleLayout.md)

UIListLayout is a class used to position sibling UI elements in rows or columns within a parent container based on `FillDirection`.

## Properties
### `UIListLayout.HorizontalFlex`
- **Type:** `UIFlexAlignment`
- **Summary:** Controls how to distribute extra horizontal space.

### `UIListLayout.ItemLineAlignment`
- **Type:** `ItemLineAlignment`
- **Summary:** In a flex layout, defines the **cross-directional** alignment of siblings within a line.

### `UIListLayout.Padding`
- **Type:** `UDim`
- **Summary:** Amount of free space between each element.

### `UIListLayout.VerticalFlex`
- **Type:** `UIFlexAlignment`
- **Summary:** Controls how to distribute extra vertical space.

### `UIListLayout.Wraps`
- **Type:** `boolean`
- **Summary:** Controls whether siblings within the parent container wrap.
