# AudioSearchParams

**Superclass:** [Instance](Instance.md)

This file defines `AudioSearchParams` class, which is used to pass parameters to the `Class.AssetService:SearchAudioAsync()` method when searching for audio assets.

## Tags
- NotReplicated

## Properties
### `AudioSearchParams.Album`

- **Summary:** The album the audio asset belongs to.
