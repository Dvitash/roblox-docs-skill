# BubbleChatMessageProperties

**Superclass:** [TextChatMessageProperties](TextChatMessageProperties.md)

BubbleChatMessageProperties is a class representing properties for the `BubbleChatMessage` object, including settings for bubble color and transparency.

## Properties
### `BubbleChatMessageProperties.BackgroundColor3`
- **Type:** `Color3`
- **Summary:** Background color of bubbles.

### `BubbleChatMessageProperties.BackgroundTransparency`
- **Type:** `double`
- **Summary:** Background transparency of bubbles.

### `BubbleChatMessageProperties.FontFace`
- **Type:** `Datatype.Font`
- **Summary:** Font of the bubble text.

### `BubbleChatMessageProperties.TailVisible`
- **Type:** `boolean`

### `BubbleChatMessageProperties.TextColor3`
- **Type:** `Color3`
- **Summary:** Color of bubble text.

### `BubbleChatMessageProperties.TextSize`
- **Type:** `int64`
- **Summary:** Size of bubble text.
