# Highlight

**Superclass:** [Instance](Instance.md)

The `Class.Highlight` class is a visual effect used to call attention to objects within an experience, allowing for customization of the highlight's silhouette and interior.

## Properties
### `Highlight.Adornee`
- **Type:** `[Instance](Instance.md)`
- **Summary:** The `Class.Instance` that the `Class.Highlight` is applied to.

### `Highlight.DepthMode`
- **Type:** `HighlightDepthMode`
- **Summary:** Controls how the `Class.Highlight` effect displays with respect to other objects in the world.

### `Highlight.Enabled`
- **Type:** `boolean`
- **Summary:** Sets whether or not the highlight is enabled.

### `Highlight.FillColor`
- **Type:** `Color3`
- **Summary:** Sets the `Datatype.Color3` value of the highlight's interior.

### `Highlight.FillTransparency`
- **Type:** `float`
- **Summary:** Sets the transparency of the highlight's interior.

### `Highlight.OutlineColor`
- **Type:** `Color3`
- **Summary:** Sets the `Datatype.Color3` value of the highlight's outline.

### `Highlight.OutlineTransparency`
- **Type:** `float`
- **Summary:** Sets the transparency of the highlight's outline.
