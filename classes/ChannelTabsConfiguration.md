# ChannelTabsConfiguration

**Superclass:** [TextChatConfigurations](TextChatConfigurations.md)

The `ChannelTabsConfiguration` class is a configuration class for setting properties of optional channel tabs in the default chat window. It is parented to `Class.TextChatService`.

## Tags
- NotCreatable

## Properties
### `ChannelTabsConfiguration.AbsolutePosition`
- **Type:** `Vector2`
- **Tags:** ReadOnly, NotReplicated
- **Summary:** Actual screen position of the channel tab bar, in pixels.

### `ChannelTabsConfiguration.AbsoluteSize`
- **Type:** `Vector2`
- **Tags:** ReadOnly, NotReplicated
- **Summary:** Actual screen size of the channel tab bar, in pixels.

### `ChannelTabsConfiguration.BackgroundColor3`
- **Type:** `Color3`
- **Summary:** Background color of the channel tabs.

### `ChannelTabsConfiguration.BackgroundTransparency`
- **Type:** `double`
- **Summary:** Background transparency of the channel tabs.

### `ChannelTabsConfiguration.Enabled`
- **Type:** `boolean`
- **Summary:** Whether to show the channel tabs.

### `ChannelTabsConfiguration.FontFace`
- **Type:** `Datatype.Font`
- **Summary:** Font used to render text in the channel tabs.

### `ChannelTabsConfiguration.HoverBackgroundColor3`
- **Type:** `Color3`
- **Summary:** Background color of a channel tab when hovering over it.

### `ChannelTabsConfiguration.SelectedTabTextColor3`
- **Type:** `Color3`
- **Summary:** Color of text in a selected tab.

### `ChannelTabsConfiguration.TextColor3`
- **Type:** `Color3`
- **Summary:** Color of text in an unselected tab.

### `ChannelTabsConfiguration.TextSize`
- **Type:** `int64`
- **Summary:** Size of the text in channel tabs.

### `ChannelTabsConfiguration.TextStrokeColor3`
- **Type:** `Color3`
- **Summary:** Color of the text stroke for text in channel tabs.

### `ChannelTabsConfiguration.TextStrokeTransparency`
- **Type:** `double`
- **Summary:** Transparency of the text stroke for text in channel tabs.
